Games

Dans ce menu vous retrouvez tous les jeux disponibles.

En vous abonnant à un jeu vous débloquez plusieurs fonctionnalités :

  • Voir les joueurs abonnés
  • Faire une demande d'ami
  • Voir les évènements publics
  • S'inscrire à un évènement
  • Créer un évènement
Amis

Le Menu 'Amis' vous permets d'avoir accès à tous vos amis.

Dans cet espace vous aurez la possibilité de :

  • Accepter une demande
  • Refuser une demande
  • Supprimer un ami
  • Accéder au chat
Mes évènements

Dans ce menu vous aurez accès à tous vos évènements.

Avec ces évènements, vous aurez alors la possibilité de :

  • Voir ceux que vous possédés
  • Voir ceux dont vous êtes abonné
  • Voir ceux qui sont en attente
  • Accepter ou refuser une invitation
  • Voir le détail d'un évènement
